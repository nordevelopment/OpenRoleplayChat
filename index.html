// Core Capabilities

🧠

Persistent RAG Memory

Powered by SQLite vector search (`sqlite-vec`). Characters automatically extract key facts from the context and recall them forever.

💬

Real-Time Streaming

Enjoy real-time streaming text (SSE) with high responsiveness. No lag, no wait, just immediate flow.

🎨

FLUX Art Engine

Generate high-resolution avatars and roleplay scenes using state-of-the-art FLUX models out-of-the-box.

🎭

Visual Customizer

Create, modify, and delete character cards directly in your browser. Configure prompts, reasoning depth, and options visually.

🛠️

Agent Toolchains

Enable agent capabilities. Characters can autonomously run tools to read/write files and paint images in a secure sandbox.

📱

Telegram Gateway

Interact with your personal character database on-the-go via the built-in private Telegram webhook adapter.

// Easy Installation

01

Install Deps

Clone the repo and run the package installer:

npm install
02

Fire Up Server

Start the app inside your terminal monitor:

npm run dev
03

Web Setup

Open browser to `http://localhost:3000`. Enter your key, click "Initialize Database", and log in!
